# 🚀 SosNow 프로젝트 기술 스택

## 🖥 프론트엔드 (Client)
- **iOS 앱**: Swift  
- **UI 프레임워크**: UIKit / SwiftUI
- **통신 방식**: REST API (HTTP, JSON 기반)

## 🛠 백엔드 (Server)
- **Spring Boot** (Java 17 이상)  
- **빌드 도구**: Gradle
- **주요 역할**: REST API 제공, 데이터 처리, DB 연동

## 🗄 데이터베이스 (Database)
- **Oracle 19c**  
- **JDBC 드라이버**: ojdbc8  

## 📱 iOS 개발 환경
- Xcode  
- Swift 5 이상  
- iOS 14 이상
- API 통신 라이브러리: Alamofire / URLSession
